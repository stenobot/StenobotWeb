Bitsynth

BitSynth is a mobile synthesizer that lets users to create music using a combination of 8-bit samples and generative synthesis.

Current version

BitSynth.com is the most recent version of BitSynth, rebuilt from the ground up as a Progressive Web App. It can be played in a browser, or installed as a mobile app for iPhone and Android.

Windows version

BitSynth for Windows is a Universal Windows Platform app. It is downloadable for free from the Microsoft Store.

Windows Phone version

BitSynth for Windows Phone is an XNA-based app designed for Windows Phone devices. It is no longer available, as the Windows Phone Store no longer exists.

ScoreKeep

ScoreKeep is a simple app for tracking scores in various games and saving the scoresheet for later. I built ScoreKeep because my family would often play long games (like Mahjong) over multiple days, but lose the paper scoresheets before we were finished. ScoreKeep automatically saves scoresheets to your browser's local storage, and lets you load them later.

Tiny Timer

Tiny Timer is a simple, stylish countdown timer app for Windows. It always stays on top of the screen. I built it to help me stay focused during work sessions. It's available for 0.99 in the Microsoft Store.
