Bitsynth

BitSynth is a mobile synthesizer that lets users to create music using a combination of 8-bit samples and generative synthesis.

Current version

BitSynth.com is the most recent version of BitSynth, built as a Progressive Web App. It can be played in a browser, or installed as a mobile app for iPhone and Android. Read about how I built it here.

Windows version

BitSynth for Windows is a Universal Windows Platform app. It is downloadable from the Windows Store. Read about how I built it here.

Windows Phone version

BitSynth for Windows Phone is an XNA-based app designed for Windows Phone devices. It is no longer available, as the Windows Phone Store no longer exists. You can read more about it here.

ScoreKeep

ScoreKeep is a simple app for tracking scores in various games and saving the scoresheet for later. I built ScoreKeep because my family would often play long games (like Mahjong) over multiple days, but lose the paper scoresheets before we were finished. ScoreKeep automatically saves scoresheets to your browser's local storage, and lets you load them later.

Mahjong Aid

Mahjong Aid is a Universal Windows App designed to teach users how to play one of my favorite games: Mahjong. It also keeps track of ongoing games. Learn more about how I built it here.
